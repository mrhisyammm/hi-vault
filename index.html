Setup Tutorial

Learn how to configure your Google Sheets backend to get the Web App URL.

1. Create Google Sheet & Apps Script

Create a new Google Spreadsheet on your Google Drive and open the script editor.

  1. Create a new Google Sheet at sheets.new.
  2. Rename the active sheet tab at the bottom-left corner to 2FA Vault.
  3. Click on the menu bar: Extensions > Apps Script. This opens the script editor in a new tab.

2. Copy & Paste Backend Code

Copy the code below, paste it into the editor, and configure your credentials.

Loading script code...
  1. Delete all default code inside the editor (usually function myFunction() { ... }).
  2. Paste the copied script code into the editor.
  3. Important: Find lines 1 & 2 in the editor: 
    var WEB_APP_USERNAME = "user";
var WEB_APP_PASSWORD = "password";
    Change these values at the very top of the editor to secure your vault!
  4. Click the Save project button (💾 Disk Icon) at the top.

3. Format the Spreadsheet

Run the setup function to format columns and set formulas automatically.

  1. Select the function setupSheet from the dropdown menu at the top (next to Debug/Run).
  2. Click the Run (▶️) button.
  3. An authorization prompt will appear. Click Review Permissions, select your Google Account, click Advanced (at the bottom), and choose Go to 2FA Vault (unsafe). Then click Allow.
  4. Verify that your Google Sheet has been automatically formatted with headings and checkboxes. You can now close the Apps Script tab.

4. Deploy as Web App

Deploy the script publicly so the Vercel app can communicate with it.

  1. In the Apps Script editor, click the blue Deploy > New deployment button at the top-right.
  2. Click the gear icon (⚙️) next to "Select type" and select Web app.
  3. Enter a description (e.g., Hi-Vault API).
  4. Execute as: Me (your email).
  5. Who has access: Anyone (this is required to connect to the spreadsheet from Vercel).
  6. Click Deploy, then copy the generated Web app URL.

5. Connect to Hi-Vault

Connect your frontend web application to the spreadsheet backend.

  1. Go to the Settings tab in Hi-Vault.
  2. Paste the Web App URL into the Web App URL input field.
  3. Click Save & Reload.
  4. Log in with the username and password you configured in Step 2!
