Setup TutorialPanduan Pengaturan

Learn how to configure your Google Sheets backend to get the Web App URL. Pelajari cara mengonfigurasi Google Sheets backend untuk mendapatkan Web App URL.

1. Create Google Sheet & Apps Script 1. Buat Google Sheet & Apps Script

Create a new Google Spreadsheet on your Google Drive and open the script editor. Buat Google Spreadsheet baru di Google Drive Anda dan buka editor skrip.

  1. Create a new Google Sheet at sheets.new.
  2. Buat Google Sheet baru di sheets.new.
  3. Rename the active sheet tab at the bottom-left corner to 2FA Vault.
  4. Ubah nama tab lembar aktif di sudut kiri bawah menjadi 2FA Vault.
  5. Click on the menu bar: Extensions > Apps Script. This opens the script editor in a new tab.
  6. Klik pada menu bar: Extensions > Apps Script (Ekstensi > Apps Script). Ini akan membuka editor skrip di tab baru.

2. Copy & Paste Backend Code 2. Salin & Tempel Kode Backend

Copy the code below, paste it into the editor, and configure your credentials. Salin kode di bawah ini, tempelkan ke editor, dan konfigurasi kredensial Anda. 

Loading script code...
  1. Delete all default code inside the editor (usually function myFunction() { ... }).
  2. Hapus semua kode bawaan di dalam editor (biasanya function myFunction() { ... }).
  3. Paste the copied script code into the editor.
  4. Tempel kode skrip yang sudah disalin ke dalam editor.
  5. Important: Penting: Find lines 1 & 2 in the editor: Cari baris 1 & 2 di editor: 
    var WEB_APP_USERNAME = "user";
var WEB_APP_PASSWORD = "password";
    Change these values at the very top of the editor to secure your vault! Ubah nilai ini di bagian paling atas editor untuk mengamankan vault Anda!
  6. Click the Save project button (disk icon) at the top.
  7. Klik tombol Save project (ikon disket) di bagian atas.

3. Format the Spreadsheet 3. Format Spreadsheet

Run the setup function to format columns and set formulas automatically. Jalankan fungsi setup untuk memformat kolom dan memasang rumus secara otomatis.

  1. Select the function setupSheet from the dropdown menu at the top (next to Debug/Run).
  2. Pilih fungsi setupSheet dari menu dropdown di bagian atas (di sebelah Debug/Run).
  3. Click the Run button.
  4. Klik tombol Run (Jalankan).
  5. An authorization prompt will appear. Click Review Permissions, select your Google Account, click Advanced (at the bottom), and choose Go to 2FA Vault (unsafe). Then click Allow.
  6. Prompt otorisasi izin akses akan muncul. Klik Review Permissions (Tinjau Izin), pilih Akun Google Anda, klik Advanced (Lanjutan) di bagian bawah, lalu pilih Go to 2FA Vault (unsafe). Kemudian klik Allow (Izinkan).
  7. Verify that your Google Sheet has been automatically formatted with headings and checkboxes.
  8. Pastikan Google Sheet Anda telah terformat secara otomatis dengan judul kolom dan kotak centang.

4. Deploy as Web App 4. Deploy sebagai Web App

Deploy the script publicly so the Vercel app can communicate with it. Deploy skrip secara publik agar aplikasi Vercel dapat berkomunikasi dengannya.

  1. In the Apps Script editor, click the blue Deploy > New deployment button at the top-right.
  2. Di editor Apps Script, klik tombol biru Deploy > New deployment (Penerapan baru) di kanan atas.
  3. Click the gear icon next to "Select type" and select Web app.
  4. Klik ikon gerigi di sebelah \"Pilih jenis\" dan pilih Web app (Aplikasi web).
  5. Enter a description (e.g., Hi-Vault API).
  6. Masukkan deskripsi singkat (contoh: Hi-Vault API).
  7. Execute as: Me (your email).
  8. Jalankan sebagai (Execute as): Me (email Anda).
  9. Who has access: Anyone.
  10. Siapa yang memiliki akses (Who has access): Anyone (Siapa saja). Ini diperlukan agar Vercel dapat terhubung.
  11. Click Deploy, then copy the generated Web app URL.
  12. Klik tombol Deploy, kemudian salin Web app URL yang dihasilkan.

5. Connect to Hi-Vault 5. Hubungkan ke Hi-Vault

Connect your frontend web application to the spreadsheet backend. Hubungkan aplikasi web frontend Anda dengan backend Google Sheets.

  1. Go to the Settings tab in Hi-Vault.
  2. Buka tab Settings di aplikasi Hi-Vault ini.
  3. Paste the Web App URL into the Web App URL input field.
  4. Tempel URL Aplikasi Web tadi ke dalam kolom input Web App URL.
  5. Click Save & Reload.
  6. Klik tombol Save & Reload.
  7. Log in with the username and password you configured in Step 2!
  8. Masuklah menggunakan username dan password yang telah Anda buat di Langkah 2!
